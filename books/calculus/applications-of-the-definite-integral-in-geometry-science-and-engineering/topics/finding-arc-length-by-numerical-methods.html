Q1. A curve is given by \(y=\sqrt{x}\) on \([1,4]\). Using the trapezoidal rule with \(n=3\) subintervals, what is the approximate arc length?

🔹 A. \(3.95\)
🔹 B. \(4.12\)
🔹 C. \(4.27\)
🔹 D. \(4.45\)

💡 Difficulty: easy | ✅ Correct: C

Q2. For the parametric curve \(x=t^2,\;y=t^3\) on \(t\in[0,2]\), which numerical method gives the most accurate estimate of arc length using only two subintervals?

🔹 A. Midpoint rule
🔹 B. Trapezoidal rule
🔹 C. Simpson's rule
🔹 D. Left‑endpoint rule

💡 Difficulty: medium | ✅ Correct: C

Q3. For the function \(y=e^{x^2}\) on \([0,1]\), which numerical method guarantees the smallest upper bound on the error when using three subintervals?

🔹 B. Right‑endpoint Riemann sum
🔹 C. Trapezoidal rule
🔹 D. Simpson’s rule
